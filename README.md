# Shopee_Intern_assessment
⭐利用Flask建立一個快速查詢Instagram上關於「#蝦皮美食」的API。

這個API的參數有：

✅q: 可以給定要查詢的字 (可為多個)  
✅n: 回應的文章數量
✅w: 每篇文章「內文」的字數

API建立完成後利用heroku部屬，獲得網址如下：  
例如：https://shopee-flask-heroku.herokuapp.com/shopee?q=檸檬&n=10&w=50  
就會回傳標題或是內文有含"檸檬"的文章中，選最多10篇文章，每一篇文章回傳最初的50個字
